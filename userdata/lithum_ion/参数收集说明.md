# 锂离子电池仿真参数收集说明

## 概述

要使用 PyBaMM 对自有配方的锂离子电池进行比较准确的仿真，需要收集以下四大类参数。本指南覆盖**普通负极**(石墨)和**复合负极**(石墨+硅碳)两种体系。

---

## 参数收集流程图 (推荐顺序)

```
┌─────────────────────────────────────────────────────────────────────┐
│ 1. 拆解电芯, 测量几何尺寸 (游标卡尺 / SEM 截面)                        │
│    ↓                                                                │
│ 2. 确定材料体系 (正极型号 / 负极型号 / 电解液配方 / 隔膜型号)           │
│    ↓                                                                │
│ 3. 制作半电池 → 测量 OCP 曲线 (GITT / 准稳态充放电)                     │
│    ↓                                                                │
│ 4. 制作半电池 → 测量扩散系数 (GITT / EIS / PITT)                       │
│    ↓                                                                │
│ 5. 制作半电池 → 测量交换电流密度 (EIS / Tafel)                          │
│    ↓                                                                │
│ 6. 测量电解液属性 (查阅文献或电导率仪 / Bruce-Vincent 法)                │
│    ↓                                                                │
│ 7. 测量热物性参数 (DSC / 激光闪射法 / Hot Disk)                         │
│    ↓                                                                │
│ 8. 测量电极片物理属性 (四探针电导率 / 压实密度 / PSD 粒径)               │
│    ↓                                                                │
│ 9. 装配全电池 → 验证模型 (1C 充放电对比)                                │
└─────────────────────────────────────────────────────────────────────┘
```

---

## 1. 电芯几何尺寸参数

**测量设备**: 游标卡尺 / 千分尺 / 扫描电镜 (SEM) 截面观察

| 参数名 | 符号 | 测量方法 | 典型值范围 |
|--------|------|----------|-----------|
| 负极集流体厚度 (铜箔) | L_cc_n | 千分尺或 SEM | 6~12 μm (消费电子更薄) |
| 负极涂层厚度 | L_n | SEM 截面 | 50~100 μm |
| 隔膜厚度 | L_sep | 千分尺或 SEM | 12~25 μm |
| 正极涂层厚度 | L_p | SEM 截面 | 50~80 μm |
| 正极集流体厚度 (铝箔) | L_cc_p | 千分尺或 SEM | 12~20 μm |
| 电极高度 (涂层宽度) | H | 直尺 | 用户电芯实际尺寸 |
| 电极宽度 (卷绕展开长) | W | 直尺 | 用户电芯实际尺寸 |
| 电芯散热面积 | A_cool | 计算外表面积 | 长×宽×面数 |
| 电芯总体积 | V_cell | 排水法或计算 | 长×宽×厚 |
| 标称容量 | Q_nom | 0.2C 或 1C 放电 | 用户电芯标称值 |
| 电流函数 | I | 测试电流 | 通常 = Q_nom (1C) |

> **消费电子提示**: 电芯尺寸小、铜/铝箔薄 (Cu 6~8 μm, Al 10~15 μm)。

## 2. 电极材料参数 — 正极

### 2.1 正极 OCP 曲线 (最重要)

| 测量方法 | 半电池 (正极 vs Li 金属) 的 GITT 或低速充放电 (C/50 ~ C/100) |
|----------|--------------------------------------------------------------|
| 设备 | 电化学工作站 (Biologic / Arbin / Neware 等) |
| 步骤 | 1. 组装正极/Li 扣式半电池 2. C/50 恒流充放 3. 取电压平均值或 GITT 准稳态电压 |
| 处理 | 用 numpy 完成函数拟合 (指数+tanh / 多项式) 或 CSV 插值 |

按正极材料体系对照参考拟合公式:

| 材料 | 拟合形态 | 参考拟合公式 (numpy 格式) |
|------|---------|-------------------------|
| LCO | 多平台 | `u = a + b*sto + c*tanh(d*(sto-e)) + ...` |
| NMC | 倾斜+小平台 | `u = a*sto + b + c*tanh(d*(sto-e)) + ...` |
| LFP | 水平平台 | `u = U0 + A*tanh(k*(sto-ε₁)) + B*tanh(k*(sto-ε₂))` |

### 2.2 正极扩散系数 D_s

| 测量方法 | GITT (恒电流间歇滴定) 或 EIS (电化学阻抗谱) |
|----------|-------------------------------------------|
| 步骤 | 在多个 SOC 点 (如每 10% SOC) 分别测量 |
| GITT 分析 | 使用 Weppner-Huggins 公式: D = (4/πτ)·(m_B·V_M/(M_B·A))^2·(ΔE_s/ΔE_t)^2 |
| EIS 分析 | Warburg 阻抗区拟合 |
| 温度依赖性 | 在多个温度 (如 10°C, 25°C, 40°C) 重复, 拟合 Arrhenius: D = D_ref · exp(-Ea/R·(1/T - 1/T_ref)) |

| 正极材料 | D_s 典型值 (m²/s) |
|----------|-------------------|
| LCO | 10⁻¹³ ~ 10⁻¹⁴ |
| NMC | 10⁻¹³ ~ 10⁻¹⁵ |
| LFP | 10⁻¹⁴ ~ 10⁻¹⁶ (且随 SOC 变化小) |

### 2.3 正极交换电流密度 i₀

| 测量方法 | EIS 在不同 SOC 点测量电荷转移电阻 R_ct |
|----------|---------------------------------------|
| 计算 | i₀ = RT / (nF · R_ct · S_a), 其中 S_a 是电极活性面积 |
| 动力学参数 | m_ref = i₀ / (c_e^α · c_s^α · (c_max - c_s)^α · arrhenius) |
| 活化能 | 不同温度下测量, Arrhenius 拟合 → E_r |

### 2.4 正极熵变 dU/dT (可逆热)

| 测量方法 | 在不同温度下 (如 10°C, 25°C, 40°C) 测量 OCP 曲线 |
|----------|---------------------------------------------------|
| 计算 | dU/dT ≈ (U(T₂, sto) - U(T₁, sto)) / (T₂ - T₁) |
| 注意 | 需要温度平衡时间 (每个温度点恒温 >2h) |
| 若不测量 | 可近似为 0 (忽略可逆热, 在低倍率仿真时影响较小) |

### 2.5 正极理化参数

| 参数 | 测量方法 | 设备 |
|------|---------|------|
| 最大锂浓度 c_max | c_max = ρ_material · Q_theoretical / (F · 3.6) | 晶体密度×理论容量计算 |
| 活性材料体积分数 | ε_am = 1 - ε_electrode (近似, 忽略粘结剂) | 压实密度/真密度计算 |
| 孔隙率 ε | 压汞法, 或 ε = 1 - ρ_压实/ρ_真 | 压汞仪 / 密度瓶 |
| 颗粒半径 r_p | 激光粒度仪 (PSD D50) | Malvern / HORIBA |
| 电导率 σ | 四探针法 | 四探针电导率仪 |
| 压实密度 ρ | 称重/体积计算 | 天平 + 千分尺 |
| 比热容 C_p | DSC (差示扫描量热) | DSC 仪 |
| 热导率 λ | 激光闪射法 (LFA) 或 Hot Disk | LFA / TPS |

## 3. 电极材料参数 — 负极 (普通石墨)

### 3.1 负极 OCP 曲线

同正极方法, 用石墨/Li 半电池的 GITT 或 C/50 充放电测量。
石墨 OCP 典型呈现多个电压平台 (staging 效应)。

### 3.2 负极扩散系数 D_s

同正极 GITT/EIS 方法。石墨扩散系数典型值: 10⁻¹³ ~ 10⁻¹⁴ m²/s。

### 3.3 负极交换电流密度 i₀

同正极 EIS 方法。

### 3.4 负极理化参数

| 参数 | 测量方法 | 典型值 |
|------|---------|--------|
| c_max | ρ×Q_theo/(F×3.6) | 石墨 ~26390 (LiC₆) ~33133 (文献非晶碳) |
| 其余 | 同正极 | |

## 4. 电极材料参数 — 复合负极 (石墨 + 硅碳, Primary + Secondary)

如果负极同时含有石墨和硅碳两相 (使用 `Custom_LCO_SiC.py` / `Custom_NMC_SiC.py` 模板), **两相需分别测量**。

### 4.1 Primary 相 (石墨)

同第 3 节的普通石墨负极, 所有参数加上 `Primary:` 前缀。

### 4.2 Secondary 相 (硅 / 硅碳)

**硅相的特殊测量注意事项**:

| 参数 | 测量方法 | 说明 |
|------|---------|------|
| **OCP 嵌锂** (`lithiation OCP`) | Si半电池慢速放电 (Li嵌入Si) | 嵌锂和脱锂方向必须分开测量 |
| **OCP 脱锂** (`delithiation OCP`) | Si半电池慢速充电 (Li脱出Si) | 硅存在显著滞后 (~0.1-0.2V) |
| **OCP 平均** | (lithiation + delithiation) / 2 | 用于初始 SOC 估算 |
| **扩散系数** | GITT/EIS (困难: 大体积膨胀) | 硅 D ~10⁻¹⁵~10⁻¹⁶, 随 Li 含量变化大 |
| **交换电流密度** | EIS | 可按石墨值×c_max_石墨/c_max_硅 缩放做初估 |
| **c_max** | 纯硅 ~278000 (Li₁₅Si₄) | 硅碳按 wt%_Si 比例折算 |
| **活性材料体积分数** | 硅在涂层的体积占比 | 5wt% Si ≈ 1.5~3 vol% (取决于碳基质) |
| **颗粒半径** | PSD (纳米级) | 纳米硅 30~200nm, 微米硅 1~5μm |

**计算硅碳复合材料的有效 c_max (以面积比容量逆推)**:

```
c_max_SiC = c_max_Si × wt%_Si / (wt%_Si + wt%_C × ρ_Si / ρ_C)
```

例如: 10wt% Si + 90wt% C, ρ_Si = 2.33, ρ_C = 2.26:
c_max_SiC ≈ 278000 × 0.10 / (0.10 + 0.90 × 2.33/2.26)
         ≈ 278000 × 0.10 / 1.028 ≈ 27043

### 4.3 占位参数快速启动 (在未测量时)

如果暂时无法测量硅相参数, 可使用以下占位值:

```python
# 硅 OCP: 使用文件中的 Mark2016 公式 (文献值)
# c_max: 按上式计算
# m_ref: m_ref_graphite × (c_max_graphite / c_max_silicon)
# D: 1e-15 m²/s
# r_p: 按供应商 PSD 数据
```

## 5. 电解液参数

### 5.1 电解液扩散系数 D_e

| 方法 | 查阅文献或使用浓度-电导率关系反推 |
|------|----------------------------------|
| 文献参考 | Nyman2008 (EC:EMC 3:7, 1M LiPF₆) 是常用的默认值 |
| 精确测量 | 受限扩散法或 PFG-NMR |
| 占位提示 | 如果不确定, 使用 Nyman2008 公式作为初值 |

### 5.2 电解液电导率 σ_e

| 方法 | 电导率仪直接测量 |
|------|-----------------|
| 步骤 | 在不同温度和浓度下测量, 拟合浓度+温度经验公式 |
| 设备 | 电导率仪 (Mettler Toledo 等) |
| 占位提示 | 使用 Nyman2008 公式作为初值: σ = 0.1297×(c/1000)³ - 2.51×(c/1000)^1.5 + 3.329×(c/1000) |

### 5.3 锂离子迁移数 t₊

| 方法 | Bruce-Vincent 极化法 |
|------|---------------------|
| 步骤 | 恒压极化 Li/Li 对称电池, 测量初始和稳态电流 |
| 计算 | t₊ = I_ss(ΔV - I₀R₀) / I₀(ΔV - I_ssR_ss) |
| 典型值 | LiPF₆ 电解液: 0.2~0.4 |
| 占位 | 如果未测量, 使用 0.26 |

### 5.4 电解液初始浓度

| 常见配方 | 浓度 (mol/m³) |
|----------|--------------|
| 1.0 M LiPF₆ | 1000 |
| 1.2 M LiPF₆ | 1200 |
| 1.0 M LiFSI | 1000 |

### 5.5 热力学因子

| 参数 | 说明 |
|------|------|
| (1 + d ln f± / d ln c_e) | 稀释溶液近似为 1.0; 浓溶液可能需要 >1 或与 c_e 相关的函数 |
| 占位 | 1.0 |
| 精确测量 | 浓差电池法 |

## 11. 简化启动方案 (参数未全部测量时)

如果暂时无法完成全部参数测量, 可采用以下优先级方案:

| 优先级 | 参数 | 不测量的代价 |
|--------|------|-------------|
| ★★★ 必须测量 | OCP 曲线 (正极+负极) | 无 OCP 无法准确预测电压 |
| ★★★ 必须测量 | 电芯几何尺寸 | 影响容量和内阻 |
| ★★ 重要 | 颗粒半径 (PSD) | 影响极化 |
| ★★ 重要 | 扩散系数 | 影响倍率性能 (用文献占位会有偏差) |
| ★★ 重要 | 电解液电导率 | 影响欧姆压降 |
| ★ 建议测量 | 交换电流密度 | 影响动力学极化 |
| ★ 建议测量 | 熵变 dU/dT | 影响热仿真 (低温/高倍率关键) |
| 可保留默认值 | SEI 参数 / LAM 参数 | 仅影响老化/衰减模型 |
| 可保留默认值 | Bruggeman 系数 (1.5) | 对常规极片影响不大 |
